# iCalculator

### Description

This is a simple clone of the iPhone calculator. It supports general operations such as `addition`, `division`, `multiplication`, and `subtraction`.

### Technologies Used

- HTML
- CSS
- Vanilla JS
- Webpack

### Installation

1. Install the packages:

   ```bash
   npm install
   ```

2. Build the project:

   ```bash
   npm run build
   ```

3. Start the application:

   ```bash
   npm run dev-server
   ```
