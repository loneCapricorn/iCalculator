# iCalculator

### Description

This is a simple clone of iPhone calculator. Here you can find general operations as `addition`, `division`, `multiplication`, `subtraction`.

### Used technologies

- HTML
- CSS
- Vanilla JS
- Webpack

### Instalation

1. Install the packages

```bash
npm install
```

2. Build

```bash
npm run build
```

3. Start the application

```bash
npm start
```
